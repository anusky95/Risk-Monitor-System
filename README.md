# RiskMonitorSystem
Informational Web application for displaying the various categories of risk based on the choices of lifestyle and the city entered by the user.

#Tools and technologies
* Tech Stack : WAMP
* Front end: HTML, CSS, Javascript
* Back end: PHP, MySQL
